# 投影&nbsp; Elevation

投影是划分结构、引导注意力的常用方式。投影以 z 轴的距离 `elevation`，来表示模块之间的关系，告知用户哪些内容是当前需要关注的。在卡片式布局中，投影也用于增强“卡片感”，来表现更明确的功能分区，让用户更轻易地理解页面。
